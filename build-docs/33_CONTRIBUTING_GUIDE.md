## Getting Started

1. Read [00_READ_FIRST.md](00_READ_FIRST.md)
2. Set up environment per [02_BUILD_ORDER.md](02_BUILD_ORDER.md) Phase 0
3. Follow [06_DEVELOPER_WORKFLOW.md](06_DEVELOPER_WORKFLOW.md)

## Pull Request Process

1. Create feature branch from develop
2. Implement changes following [04_CODING_STANDARDS.md](04_CODING_STANDARDS.md)
3. Add tests per [09_IMPLEMENTATION_CHECKLISTS.md](09_IMPLEMENTATION_CHECKLISTS.md)
4. Run verification: cargo test, cargo clippy, cargo fmt
5. Fill out PR template
6. Request review from domain experts

## Code of Conduct

- Be respectful and constructive
- Focus on the code, not the person
- Assume positive intent
- Document decisions
