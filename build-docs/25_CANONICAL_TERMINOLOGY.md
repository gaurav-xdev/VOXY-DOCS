## Purpose

Ensures consistent terminology across all VOXY artifacts: code, documentation, UI, marketing, and support.

---

## Terminology Standards

| Preferred Term | Avoid | Context |
|---------------|-------|---------|
| Voice command | Voice instruction, verbal order | User interaction |
| Wake word | Trigger word, hotword | Activation |
| Transcription | Transcript (as verb) | ASR output |
| Intent | Command type, action type | NLU output |
| Entity | Parameter, slot | NLU extraction |
| Local AI | On-device AI, edge AI | Architecture |
| Cloud fallback | Online mode, remote processing | Connectivity |
| Action | Command execution, task | Action Engine |
| Plugin | Extension, add-on | Extensibility |
| Workspace | Project folder, working directory | File system |

---

## Prohibited Terms

| Term | Reason | Replacement |
|------|--------|-------------|
| Hey Voxy (hardcoded) | Configurable wake word | Wake word or configured name |
| AI Assistant (in UI) | Too generic | VOXY or voice companion |
| Upload | Implies cloud | Share (with consent) |
| Sync | Implies cloud data transfer | Update or Refresh |
